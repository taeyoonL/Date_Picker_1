# App created using ***Date Picker View*** and ***Image View***

# 주요 기능
1. 현재 날짜와 시각을 상단에 띄워서 자세히 알려준다.
2. 밑에서 원하는 날짜와 시각을 선택할 수 있다.
3. 선택한 날짜와 시각이 현재 날짜, 시각과 일치할 때 화면 속 전구에 불이 들어오고 1분 동안 꺼지지 않는다.
